# Welcome to DoxaVue

DoxaVue is a Vue 3 component library built on top of the Doxa design system.

## Installation

```bash
npm install doxavue
```

## Usage

Import components and use them in your Vue app:

```vue
<script setup>
import { DXButton } from 'doxavue'
</script>

<template>
  <DXButton>Click me</DXButton>
</template>
```
